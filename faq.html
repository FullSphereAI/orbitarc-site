FAQ

What do I need to run Orbit Arc?

A Claude Pro subscription (or higher), the Claude desktop app, and one folder on your machine. That's the complete list. No terminal tools, no packages, no new hardware.

Does it work on Mac and Windows?

Yes. The core system runs on Mac and Windows through the Claude desktop app. One advanced component, voice capture, is macOS only, and the documentation says so up front.

Is this another subscription?

No. One-time purchase. The skills run inside the Claude subscription you already pay for and carry no fees of their own.

Do I need a dedicated machine?

No. The core system runs on the laptop or desktop you already use. A dedicated always-on machine is an optional advanced path for background automation, documented in the included methodology, for if and when you outgrow the core setup.

How long does setup take?

The included Quick Start is built around your first 60 minutes: create a folder, build four files, connect, boot, install skills, run three spot-tests. The full documentation covers every step in detail, and every major stage ends with a gate check so you know it actually worked.

Can I rename the assistant?

Yes. The default identity ships ready to run, but the name, persona, and tone are yours to define. The architecture doesn't care what you call it.

What exactly do I get?

31 installable skills, the complete memory architecture, the boot system, and nine designed guides: Quick Start, full setup documentation, the consolidated manual, the skills guide, FAQ and troubleshooting, the Windows setup companion, plus the advanced-layer specs for always-on automation, dedicated-machine migration, and voice capture (macOS). Every guide ships in markdown plus a designed browser edition.

What makes this different from a mega-prompt or a template pack?

Persistence and state. Prompts reset every session. Orbit Arc's memory files persist on your machine, get read before every response, and get written back as your state changes. The skills are triggered workflows, not paste-in text. It behaves like an operator because it keeps state like one.

What if I don't want to set it up myself?

That's an option. I set systems up with you remotely over a screen share, or for you on-site if you're local. No pitch, no pressure: email uplink@orbitarc.co and I'll tell you honestly whether this fits what you're trying to do.
